Python is widely used for general-purpose and high level programming language. Python was planned by Guido van Rossum in 1991 and it was further developed by Python Software Foundation. It was mainly developed for emphasis on code readability, its syntax allows programmers to express concept in fewer lines of code.

Python 2 and Python 3 have different (sometimes incompatible) libraries

Many of today's developers are creating libraries strictly use with Python 3, many older libraries built for Python 2 are not compatible.
You may be able to port a 2.x library to 3.x., this can be difficult and complicated.

It is used for..

• Python can be used on a server to create web applications.
• Python can be used alongside software to create workflows
• Python can connect to database systems. Python can also read and modify files.
• • Python can be used to handle big data and perform complex mathematics.
• Python can be used for rapid prototyping for production-ready software development.

What can Python do?

• Python works on different platforms like: Windows, Mac, Linux, Raspberry Pi, etc
• Python has simple syntax similar to the English language.
• Python has syntax that allows developers to write programs with fewer as compared to other programming languages.
• Python runs on an interpreter system (code can be executed as soon as it is written). This means that prototyping can be very quick.
• Python can be treated in a procedural way, an object-orientated way or a functional way.

Why Python?

Comparison with other Languages:
Python is often compared to other languages such as Java, JavaScript, Perl, Tcl, or Smalltalk. Comparisons to C++, Common Lisp and Scheme can also be enlightening. The choice of a programming language is often dictated by other real-world constraints such as cost, availability, training, and prior investment, or even emotional attachment. Since these aspects are highly variable, it seems a waste of time to consider them much for this comparison.At some point in time, we had almost all the programming languages we could count on our fingers. Today there are many and all with their specialties. But what make a language unique are its features. And, ultimately, its features make it be chosen or approved for a project.

FEATURES OF PYTHON PROGRAMMING:
EASY TO LEARN AND USE: Python is very easy to code as compared to other popular languages like C, C++ and JAVA. We can learn Python syntax in just a few hours. Python is safe, mastering it requires learning all its advanced concepts, packages and modules, which takes time therefore, it is easy to program. Since it is written dynamically, it requires a recess, therefore it is readable.
EXPRESSIVE LANGUAGE: Suppose we have two languages, A and B, and all the programs that can be done in A can be done in B using local transformations.
INTERPRETED LANGUAGE: If you are familiar with languages like C, C ++ or Java, you have to first compile the code and then run it, but in Python it is not necessary to compile the code. The source code becomes an immediate form called bytecode. In Pythob, all you need to do is run your Python code without worrying about linking to libraries and other things. The source code is executed line by line, and not all at once, therefore it is easier to debug the code. The interpretation makes it a little bit slower than Java, but that does not matter with the advantages offered.

CROSS PLATFORM LANGUAGE:
Suppose you have written a Python code for your Windows machine and if you want to run it on Linux, Unix and Macintosh you don’t need to make any changes in the code. In other words, you can run the code on any machine, it is not necessary to write a different code for different machines, therefore Python is a portable language.

FREE AND OPEN SOURCE:
The Python language is available for free on the official website and it is open source. It means source code is available to the public for free. And you can download the code, change it, use it, therefore it is open source.

LARGE STANDARD LIBRARY:
Python has a large and extensive library which offers a wide set of modules and functions for rapid application development to Users.

EXTENSIBLE:
You can write some of the Python codes in other languages such as C, C ++. Therefore, Python is an extensible language, which means it can be extended to other languages.

GUI PROGRAMMING SUPPORT:
We can develop Graphical user interfaces applications using Python.

OBJECT-ORIENTED LANGUAGE:
The Programming language that can model the real world is called object-oriented language. Python focuses on objects and combines data and functions. A language oriented to the procedure revolves around the functions, which are codes that can be reused. Python supports procedural and object-oriented programming language, which is the key feature of Python. It also supports multiple inheritance, unlike Java.

DYNAMICALLY TYPED:
Python is a dynamically typed language. It means that type for a value is decided at runtime, not in advance. It is not necessary to specify the type of data when declaring it.
